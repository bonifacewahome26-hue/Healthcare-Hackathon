# Healthcare-Hackathon
Code Repository for Heatmap Hackathon
Team 18: Code Repository for Heatmap Hackathon

.RMD and .PDF Key Findings This file includes the spatial maps used for key findings in the team presentation. The LEGEND for these maps can be found in the team slides. Plots trauma and burn capacity in nortwestern U.S. and Montana by adult vs pediatric capacity. Utilizes NIRD data.

Excel Data Dashboard The first sheet of the Excel spreadsheet contains our team's interactive Data Dashboard. Additional sheets in the Excel file are for functionality of the dashboard. This dashboard looks at burn bed distribution throughout the United States by region and state. This dashboard also looks at adult versus pediatric burn capacity and population densities in comparison to burn bed distribution. Utilizes NIRD data and U.S. Census data.

.RMD and .PDF Exploratory Spatial Analyses This file includes spatial analyses for the broader northwestern U.S. region. Maps plot burn and trauma centers by adult versus pediatric capacity, by LVI vs LVII, and by type of verified burn designation (ABA, state-designated, NA). Key findings maps in the slide presentation are NOT included in this file. Utilizes NIRD data.

.RMD and .PDF Montana vs Maine Comparisons This file includes comparative bar graphs for healthcare infrastracture in Maine and Montana external to trauma and burn centers. Utilizes the sources linked in the .rmd and .pdf.
